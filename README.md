# MarketMind AI

MarketMind AI is a simple AI-powered assistant designed to help small traders make better business decisions using minimal data.

## Features

- Sales summary
- Demand prediction
- AI-powered advice
- Simple and user-friendly interface

## Tech Stack

- Python
- Streamlit
- Pandas
- OpenAI API

## Run the App

```bash
pip install -r requirements.txt
streamlit run app.py
